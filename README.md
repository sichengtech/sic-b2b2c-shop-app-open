# 项目介绍
思程 Shop App 的工程名称是 SiC B2b2cShop App Open ，他是思程 Shop的移动端(买家端)。 采用uni-app框架开发，一次开发可产出：安卓App端、IOS App端、H5端、小程序端。  

# 开源地址
